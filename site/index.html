About

Welcome to Darkweb Kittens. As you may have guessed, this is a site about kittens. And it's on the "dark web".

This site is largely a project for me to experiment with web development and server configs on a site that isn't going to matter when I end up breaking stuff.

The site is inspired by Ethan Zuckerman's Cute Cat Theory of digital activism. The theory suggests that when more people start using a particular service (in this case Tor Onion Services) for mundane activities, like sharing kitten pictures, the resistance of that service to government interference or takedown also increases.

Tor Onion Services

This site is available as a Tor Onion Service at:

7h4hhpykscqbstlboqey4dbncrxd2cjbtfs3bnljvopcn4h6by4ngiad.onion

Onion Services are special web addresses that are understood by the Tor anonymity software and can be accessed using the Tor Browser.

Onion domains provide a number of security benefits over regular domains, such as:
 •  Additional privacy
 •  Resistance to surveillance
 •  Resistance to censorship
 •  Enhanced integrity/tamperproofing
 •  No registration or central authority

If you visited this site using Tor, you're were probably already redirected to the onion address! For more info on how I achieve this, see the OpenBSD tab.

OpenBSD

This site is currently served from a VPS running OpenBSD and uses the native httpd as a webserver, along with pf as a firewall. Both packages are included in the base install of OpenBSD. Only one aditional package — tor — has been installed on the system.

httpd

The base installation of OpenBSD ships with httpd, a lightweight and full-featured secure webserver. (Not to be confused with the similarly-named Apache httpd).

httpd is an incredibly powerful webserver which can be surprisingly easily configured. It uses the OpenBSD project's LibreSSL to handle encryption.

For more info on httpd, I highly recommend this book by Michael W Lucas.

pf

Also available out-of-the-box in the base install of OpenBSD, pf is recognised as one of the most powerful firewall packages in the industry.

The pf firewall is the magic sauce behind this site's automatic onion redirect. See here to find out how I implemented it.
